# Wolf3D Map to Obj

**This project is a work-in-progress.**

A script to export Wolfenstein 3D maps to OBJ object files.

## Getting Started

### Prerequisites

You need Python 3.6 or higher to run this.

### Installing

Clone or download this repository.

    pip install -r requirements.txt
